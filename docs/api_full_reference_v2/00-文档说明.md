- 优先保证“路径、方法、鉴权、中间件、注册位置、是否支持 admin-key”准确。
- 对旧版已经明显落后的部分，直接按当前路由重排，而不是继续在旧内容上打补丁。
- 不再强制为每个 endpoint 展开完整 body 字段；若需要字段级请求参数，请继续查对应 handler、DTO 与 service。
